General Information

Online Play is a feature in Project Gamblitz allowing mod users to play online with or against other users.

Project Gamblitz server

The mod uses a LAN server in order to handle the Online Play status view, kicks, stats and other stuff. This information is sent by your console in the background and can be seen in the Project Gamblitz Discord Server.

"Gamblitz" Identifier

In order to tell your console appart from others, a unique console ID is used through all Project Gamblitz Server communications. This ID consists of 32 randomized symbols from 0 to 9 and A to Z. The ID is completely random and it cannot be repeated (since it is randomly generated, it cannot be used in reverse to obtain sensitive data of your console). You can treat the "Gamblitz" Identifier as your "friend code" specific for the mod.

Online Play Rules

Online Play has a series of rules so everybody can play safely and enjoy the game:

  1. Names containing profanity, insults or any other kind offensive, inappropiate or annoying words are not allowed.
    • This also includes impersonation of other users.
  2. Repeatedly using exploits that are disruptive or give you advantage over other players is not allowed. If you find such exploit please report it in the Project Gamblitz server. For example:
    • Using hacks or cheats (instant perma-ban).
    • Using any sort of external tool that can interfere with the gameplay (instant perma-ban).
    • Using unintended glitches that give advantage.
    • Teaming with other users to get strategical advantage, unless fomented by the current game mode or ongoing event.
  3. Trolling is not allowed unless it is fomented by the current game mode or gives you a strategical advantage. For example:
    • Do not have a toxic attitude against other players.
  4. Editing any of the mod files is not allowed and will get you banned:
    • Such as Mush, Params, etc. edits. We have an Anti-cheat which will detect if any modification has been made to the mod and will block you from connecting to the server.
If you are found breaking any of this rules you will be given a warning, be temporarily banned from Online Play. While permanent bans are rare, repeated severe offenses will cause them.

Privacy Notice about Telemety

All the activity and requests your console does related to Online Play is logged (such as online matches). This is done so we can properly moderate Online Play and handle custom Splatfests. The logged contents include your "Gamblitz" Identifier and Mii/Account name. By executing the mod, you agree that the Staff members can see this information (which will be always kept private).
